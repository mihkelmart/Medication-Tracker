Google Sheets sync
Step 1 — create your sheet

Create a new Google Sheet at sheets.google.com. The header row will be written automatically on your first save.

Step 2 — add the Apps Script relay

In your sheet, go to Extensions → Apps Script. Delete any existing code and paste the script below. Then click Deploy → New deployment → Web app. Set "Who has access" to Anyone and click Deploy. Copy the URL it gives you.
